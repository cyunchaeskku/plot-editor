# 수정 요청 사항
1. 배경 색
- 전체적인 배경 색은 은은한 light grey로 사용하도록 수정
2. 에디터 기능
- '/' 입력하면 여러 서식이 나오는데, 여기서 화살표를 이용해 선택할 수 있도록.
- 그리고 만약 현재 '안나'라는 인물의 대사 서식이었고 Enter를 클릭하면 다음 줄도 '안나'의 대사 서식이 되어야 함. 여기서 아무 것도 입력하지 않고 한 번 더 Enter 클릭하면 기본 서식인 '단락'으로 변경되도록. 직전 line의 서식이 유지되도록 한다.
3. 인물 상세
- 현재 **관계**는 입력할 수 있지만 인물의 **특성**은 입력할 수 없음.
  - 이 인물이 어떤 인물인지 (e.g., 부자임, 연약한 성격)를 각각의 항목으로 입력할 수 있도록 수정
4. 관계도 고도화
- 각 인물(node)는 circle로.
- relation은 directed arrow로 표시되도록.
  - 그리고 어떤 relation인지도 표시되도록.
