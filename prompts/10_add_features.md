# New Features to add
## novel editor
- 지금까지 만든 것은 plot editor이다.
- 거기에 더해 novel editor를 추가한다.
- novel editor는 단순 줄글을 작성하는 것으로, Microsoft Word에서 글을 작성하는 것을 생각하면 쉬울 것이다.
- plot editor, novel editor는 작품 별로 선택할 수 있게 한다 (A 작품은 플롯 에디터, B 작품은 novel editor로 사용 등)
- 다른 기능은 모두 동일하게 사용한다. (인물 상세, 관계도, 기획서)
- 기획서 에디터는 markdown editor 느낌이 강한데, novel editor는 microsoft word 느낌의 전문적 줄글 editor가 되었으면 좋겠다.
- novel editor에서는 tab을 누르면 들여쓰기가 가능해야 한다. (단락의 첫 시작은 들여쓰기임)
