# 게시판 기능 추가
- 지금까지는 하나의 페이지에서 개발을 했지.
- 이제 새로운 페이지가 필요해. 그것은 게시판 페이지야. (e.g., /thread 등 게시판 링크 이름은 알아서 잘 지어줘)
- 사용자가 작성해서 dynamodb와 s3 bucket에 저장된 정보들을 사용자들이 서로 공유하고, 내용을 확인하고, 댓글(feedback)을 달 수 있는 online 게시판을 만들 거야.
- 일단 UI를 먼저 디자인하고 싶어. 실제로 s3 bucket을 연결하지는 말고, 현재 어떤 내용들이 저장되는지 확인하고, 프론트엔드 UI도 확인한 뒤 사용자들이 자신의 작품을 공유하고 댓글을 달고 할 수 있는 게시판 페이지를 먼저 디자인해봐. dummy data를 넣어서 만들어봐. 실제 AWS backend는 아직 연결 안 할 거니까
- 디자인 할 때에는 ui-designer agent를 사용해서 예쁘고 세련된 페이지를 만들어줘. **주의** 배경색과 글자색이 비슷해서 글자가 잘 안 보이게 되는 경우는 꼭 방지해줘.
